# 資産管理システム（研修用）

## 概要
会社の資産（PCなど）を管理するWebアプリケーション  
:contentReference[oaicite:0]{index=0} を用いたCRUD + セッション管理の学習を目的とする

---

## 主な機能
- ログイン（セッション管理）
- 資産一覧表示
- 資産登録
- 貸出管理（課題）
- ユーザ管理

---

## 画面一覧

| 画面 | URL | 説明 |
|------|-----|------|
| ログイン | `/` | ユーザ名入力 |
| メニュー | `/menu` | 機能選択 |
| 資産一覧 | `/assets` | 資産一覧表示 |
| 資産登録 | `/assets/new` | 新規登録 |
| 貸出管理 | `/loans` | 貸出状況 |
| ユーザ管理 | `/users` | ユーザ一覧・登録・削除 |
| パスワード設定 | `/user/setup-password` | ユーザのパスワード変更 |
