# 資産管理システム（研修用）

## 概要
会社の資産（PCなど）を管理するWebアプリケーション  
Spring Boot を用いたCRUD + セッション管理の学習を目的とする

---

## 主な機能
- ログイン/ログアウト機能（セッション管理）
- 資産情報の一覧表示、登録、更新、削除 (CRUD)
- 貸出情報の一覧表示、登録、更新、削除 (CRUD)
- ユーザ情報の一覧表示、登録、更新、削除 (CRUD)

---

## 画面一覧

| 画面 | URL | 説明 |
|------|-----|------|
| ログイン | `/` | ログイン画面。ユーザ名とパスワードで認証します。 |
| メニュー | `/menu` | ログイン後に表示されるメニュー画面。各機能へ遷移します。 |
| 資産一覧 | `/assets` | 登録されている資産情報を一覧で表示します。 |
| 資産登録 | `/assets/new` | 新しい資産情報を登録します。 |
| 貸出管理 | `/loans` | 資産の貸出状況を一覧で表示します。 |
| ユーザ管理 | `/users` | ユーザ情報を一覧で表示・登録・削除します。 |
| パスワード設定 | `/user/setup-password` | ユーザのパスワードを変更します。 |

---

## 使用技術
- **フレームワーク:** Spring Boot 3.3.0
- **言語:** Java 21
- **テンプレートエンジン:** Thymeleaf
- **データベース:** H2 (開発用), PostgreSQL (本番/演習用)
- **ORマッパー:** Spring Data JPA
- **ビルドツール:** Maven
