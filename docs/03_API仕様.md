# API仕様

## 認証
### ログイン
- `POST /login`
- ユーザ名とパスワードで認証する。
- **リクエストパラメータ:**
    | パラメータ | 型 | 説明 |
    |-----------|----|------|
    | username  | String | ユーザ名 |
    | password  | String | パスワード |

### ログアウト
- `POST /logout`
- セッションを破棄してログアウトする。

---

## 資産
### 資産一覧
- `GET /assets`
- 登録済みの資産を一覧で取得する。

### 資産登録画面
- `GET /assets/new`
- 資産を登録するための画面を返す。

### 資産登録処理
- `POST /assets/new`
- 新しい資産を登録する。
- **リクエストパラメータ:**
    | パラメータ | 型 | 説明 |
    |-----------|----|------|
    | name      | String | 資産名 |

---

## 貸出
### 貸出一覧・登録画面
- `GET /loans`
- 貸出一覧と、貸出可能な資産、ユーザの一覧を画面に表示する。

### 貸出入力画面
- `GET /loans/new`
- **リクエストパラメータ:**
    | パラメータ | 型 | 説明 |
    |-----------|----|------|
    | assetId   | Long | 貸出対象の資産ID |

### 貸出処理
- `POST /loans`
- 資産をユーザに貸し出す。
- **リクエストパラメータ:**
    | パラメータ | 型 | 説明 |
    |-----------|----|------|
    | assetId    | Long | 資産ID |
    | userId     | Long | ユーザID |
    | loanDate   | String | 貸出日 (yyyy-MM-dd) |
    | periodDays | Integer| 貸出期間（日数） |

### 返却処理
- `POST /loans/return`
- 貸出中の資産を返却する。
- **リクエストパラメータ:**
    | パラメータ | 型 | 説明 |
    |-----------|----|------|
    | loanId     | Long | 返却対象の貸出ID |

---

## ユーザ
### ユーザ一覧・登録画面
- `GET /users`
- 登録済みのユーザを一覧で表示し、新規ユーザ登録フォームを提供する。

### ユーザ登録処理
- `POST /users`
- 新しいユーザを登録する。
- **リクエストパラメータ:**
    | パラメータ | 型 | 説明 |
    |-----------|----|------|
    | name      | String | ユーザ名 |
    | password  | String | 初期パスワード |

### ユーザ削除処理
- `POST /users/delete`
- ユーザを削除する。
- **リクエストパラメータ:**
    | パラメータ | 型 | 説明 |
    |-----------|----|------|
| id        | Long | 削除対象のユーザID |

### パスワード設定画面
- `GET /user/setup-password`
- ユーザのパスワードを設定するための画面を返す。
- **リクエストパラメータ (任意):**
    | パラメータ | 型 | 説明 |
    |-----------|----|------|
    | userId    | Long | パスワード変更対象のユーザID |

### パスワード設定処理
- `POST /user/setup-password`
- ユーザのパスワードを更新する。
- **リクエストパラメータ:**
    | パラメータ | 型 | 説明 |
    |-----------|----|------|
    | userId    | Long | 対象ユーザID |
    | password  | String | 新しいパスワード |

### 権限変更処理
- `POST /users/update-role`
- ユーザの権限を Admin に変更する。
- **リクエストパラメータ:**
    | パラメータ | 型 | 説明 |
    |-----------|----|------|
    | id        | Long | 対象ユーザID |
