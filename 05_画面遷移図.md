# 画面遷移図

## 1. 全体フロー

```mermaid
stateDiagram-v2
    [*] --> LoginView : アクセス

    state "ログイン画面 (/)" as LoginView {
        [*] --> Input
        Input --> Authenticate : ログインボタン
        Authenticate --> Input : 失敗
    }

    state "メニュー画面 (/menu)" as MenuView {
        [*] --> SelectFunction
    }

    state "資産管理 (/assets)" as AssetView {
        [*] --> AssetList
        AssetList --> AddAsset : 登録ボタン
        AddAsset --> AssetList : リダイレクト
    }

    state "貸出管理 (/loans)" as LoanView {
        [*] --> LoanList
        LoanList --> AddLoan : 貸出ボタン
        AddLoan --> LoanList : リダイレクト
        LoanList --> ReturnLoan : 返却ボタン
        ReturnLoan --> LoanList : リダイレクト
    }

    state "ユーザー管理 (/users)" as UserView {
        [*] --> UserList
        UserList --> AddUser : 登録ボタン
        AddUser --> UserList : リダイレクト
        UserList --> DeleteUser : 削除ボタン
        DeleteUser --> UserList : リダイレクト
    }

    LoginView --> MenuView : 認証成功
    MenuView --> LoginView : ログアウト

    MenuView --> AssetView : 資産管理へ
    AssetView --> MenuView : 戻る

    MenuView --> LoanView : 貸出管理へ
    LoanView --> MenuView : 戻る

    MenuView --> UserView : ユーザー管理へ
    UserView --> MenuView : 戻る
```

## 2. 画面詳細

| 画面名 | URL | 主なアクション | 遷移先 |
| :--- | :--- | :--- | :--- |
| **ログイン** | `/` | ログイン | 成功: メニュー<br>失敗: 自画面 |
| **メニュー** | `/menu` | 機能選択<br>ログアウト | 各機能画面<br>ログイン画面 |
| **資産一覧** | `/assets` | 新規登録<br>戻る | 自画面 (更新)<br>メニュー |
| **貸出管理** | `/loans` | 貸出登録<br>返却<br>戻る | 自画面 (更新)<br>自画面 (更新)<br>メニュー |
| **ユーザー管理** | `/users` | ユーザー登録<br>削除<br>戻る | 自画面 (更新)<br>自画面 (更新)<br>メニュー |
